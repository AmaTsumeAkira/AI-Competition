# CLI部署 Level 1：Python 网页数据采集与分析

## 任务说明

编写一个 Python 脚本，从指定网页抓取数据并保存为 CSV 文件，同时实现数据统计与筛选功能。

**考核目标：** 使用 AI 命令行工具（如 Claude Code、Codex CLI）完成编码任务。

## 安装依赖

```bash
# Windows PowerShell
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple/ requests beautifulsoup4 pandas

# Mac/Linux
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple/ requests beautifulsoup4 pandas
```

## 任务要求

### 基础任务（10分）
1. 实现 `print_system_info()` —— 打印 Python 版本、操作系统、工作目录、库版本等环境信息
2. 从 `https://quotes.toscrape.com/` 抓取名言数据
3. 提取：名言内容（text）、作者（author）、标签列表（tags）
4. 实现翻页抓取，至少抓取 3 页数据（≥30 条）
5. 保存为 `quotes.csv`，UTF-8 编码，含表头

### 统计分析任务（4分）
6. 实现 `get_top_authors(quotes, n)` 函数：统计引用次数最多的前 n 位作者
7. 实现 `filter_by_tag(quotes, tag)` 函数：筛选包含指定标签的名言

### 进阶任务（6分）
8. 支持命令行参数：`--output` 指定输出文件名，`--tag` 指定标签过滤
9. 运行脚本后在终端打印统计摘要（总条数、作者数、Top 3 作者）
