# AI纠错 Level 2：多文件·接口陷阱

## 任务说明

你收到了一个 Flask 任务管理微服务项目，包含 3 个 Python 文件。项目可以正常启动但运行结果不正确，部分功能失效。

**你的任务：** 使用 AI 工具定位并修复所有 Bug，使项目通过自动化测试。

## 文件清单

| 文件 | 说明 |
|------|------|
| `app.py` | Flask 主应用（~150 行） |
| `models.py` | 数据模型（~60 行） |
| `utils.py` | 工具函数（~80 行） |
| `test_runner.py` | 自动化测试（选手勿改） |

## 已知问题

- 任务创建时日期解析异常
- 分页功能返回结果不正确
- 错误处理机制有问题
- 批量导入功能异常

## 评分标准

- **测试通过率**：60%（15 个测试用例，通过 ≥10 个满分）
- **修复日志质量**：20%（`fix_log.md`：根因分析 + 修复策略）
- **过程分**：20%（终端录屏：工具选择多样性 + 回归验证）
- **满分**：35 分
- **限时**：60 分钟
- **附加分**：fix_log.md 质量达标 +5 分

## 提交物

1. 修复后的全部源代码
2. `fix_log.md`（Bug 清单 + 根因 + 修复方案）
3. 终端操作录屏
